# Rust 101

Rust 101 is an introduction to Rust programming, with a focus on topic summaries that we believe can be especially helpful for newcomers to Rust.

Rust 101 aims to explain the topics in a broad way first. We recommend reading Rust 101 along with the official Rust documentation book: [The Rust Programming Language](https://doc.rust-lang.org/book/).

Rust 101 is created by SixArm and Joel Parker Henderson, with content generated by OpenAI ChatGPT using explain prompts. We welcome constructive feedback, and we're improving this repo often.
